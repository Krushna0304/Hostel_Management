## 🎨 Design System

### Colors
```css
--primary: #0F172A (Dark slate)
--accent: #38BDF8 (Sky blue)
--success: #10B981 (Green)
--warning: #F59E0B (Amber)
--danger: #EF4444 (Red)
```

### Typography
- **Font Family:** Inter (Google Fonts)
- **Headings:** 800 weight
- **Body:** 400-600 weight
- **Sizes:** Responsive scale

### Spacing
- **Sections:** 100px padding
- **Cards:** 32px padding
- **Gaps:** 16-48px

### Shadows
- **Small:** Subtle elevation
- **Medium:** Card hover
- **Large:** Featured elements
- **XL:** Hero mockup

### Border Radius
- **Small:** 8px (buttons)
- **Medium:** 12-16px (cards)
- **Large:** 20px (badges)

## 🛠️ Customization Guide

### Changing Colors
Update CSS variables in `:root`:
```css
:root {
    --primary: #YourColor;
    --accent: #YourColor;
}
```

### Adding Sections
1. Create HTML structure
2. Add corresponding CSS
3. Update navigation links
4. Add scroll animations

### Modifying Pricing
1. Update pricing cards in HTML
2. Adjust grid columns if needed
3. Update feature lists
4. Change CTA links
